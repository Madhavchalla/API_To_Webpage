# 3D Product Showcase

A stunning React application that displays an interactive 3D product gallery fetched from the Fake Store API. Features smooth animations, mouse-following tilt effects, and a sleek dark theme.

https://productcardsapi.netlify.app/

## ✨ Features

- **3D Interactive Cards**: Mouse-following tilt animations with depth effects
- **Smooth Animations**: Powered by Framer Motion for fluid transitions
- **Responsive Design**: Optimized for desktop and mobile devices
- **Dark Theme**: Elegant black gradient background with glassmorphism cards
- **Real-time Data**: Fetches products from Fake Store API
- **Scrollable Descriptions**: Full product descriptions with custom scrollbars

## 🛠️ Tech Stack

- **React** - UI framework
- **Framer Motion** - Animation library
- **CSS3** - Styling with modern features
- **Fake Store API** - Product data source
